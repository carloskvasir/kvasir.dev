# Exemplo de Post com Imagem

Este é um exemplo simples de como adicionar imagens aos seus posts.

## Como Funciona

As imagens ficam na mesma pasta que o arquivo MDX do post:

![Estrutura de pastas](./folder-structure.png.placeholder)

## Vantagens desta Abordagem

- **Organização**: Tudo relacionado ao post fica junto
- **Portabilidade**: Fácil de mover ou fazer backup
- **Simplicidade**: Caminhos relativos são mais fáceis de gerenciar
- **Versionamento**: Git rastreia imagens junto com o conteúdo

## Exemplo de Uso

Basta usar a sintaxe padrão do Markdown com caminho relativo:

```markdown
![Descrição da imagem](./nome-da-imagem.png)
```
